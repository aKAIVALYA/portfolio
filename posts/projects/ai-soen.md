AI-SOEN is a browser-native AI collaborative IDE designed to support real-time multi-user coding with zero-install execution.

### Key Features:
* **WebContainer Integration:** Powered by WebContainers to run a complete, secure NodeJS environment entirely within the user's browser.
* **Real-time Collaboration:** Leverages Socket.io and Redis for instant multiplayer code synchronizations.
* **AI Coding Assistant:** Integrated AI chatbot with contextual code understanding and secure sandbox workspace generation.
