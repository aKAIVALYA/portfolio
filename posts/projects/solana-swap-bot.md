An automated Solana trading bot that monitors target users on X (formerly Twitter) to extract token signals and execute swaps.

### Key Features:
* **X Sentiment Analysis:** Utilizes GPT-4o to parse posts from selected profiles in real-time, executing high-speed swaps upon detecting bullish signals.
* **On-Chain Transactions:** Integrates Raydium protocol via Solana Web3.js SDK for secure, automated transaction processing.
* **Environment Security:** Features secure environment-based wallet and private key configuration management.
